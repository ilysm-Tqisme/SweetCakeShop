# 🍰 SweetCakeShop - Website Bán Bánh Trực Tuyến

SweetCakeShop là đồ án phát triển từ một dự án website bán bánh có sẵn và được mở rộng thêm nhiều chức năng nhằm nâng cao trải nghiệm người dùng cũng như hỗ trợ quản trị hệ thống hiệu quả hơn.

Trong quá trình thực hiện, mình tập trung nghiên cứu và phát triển các tính năng ứng dụng AI, thống kê doanh thu và quản lý khuyến mãi, đồng thời cải thiện một số chức năng của hệ thống để phù hợp với mô hình thương mại điện tử hiện đại.

---

# 🚀 Công nghệ sử dụng

* **Ngôn ngữ:** C#
* **Framework:** ASP.NET Core MVC (.NET)
* **Database:** SQL Server
* **ORM:** Entity Framework Core
* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **AI:** RAG (Retrieval-Augmented Generation)
* **Xuất báo cáo:** Excel, PDF
* **IDE:** Visual Studio 2022
* **Version Control:** Git & GitHub

# 🤖 Các chức năng được phát triển thêm

Đây là những chức năng mình đã nghiên cứu và tích hợp vào dự án:

* 🤖 **Chatbot AI sử dụng mô hình RAG (Retrieval-Augmented Generation)** giúp tìm kiếm thông tin và trả lời dựa trên dữ liệu của hệ thống.
* 👥 **Hai chatbot riêng biệt** dành cho **User** và **Admin**, đáp ứng các nhu cầu sử dụng khác nhau.
* 📊 **Thống kê doanh thu**, hỗ trợ theo dõi tình hình kinh doanh trực quan.
* 📄 **Xuất báo cáo doanh thu** dưới định dạng **Excel** và **PDF**.
* 🎁 **Hệ thống mã giảm giá (Voucher/Coupon)** áp dụng khi thanh toán.
* ⚡ Cải thiện và tối ưu một số chức năng của hệ thống nhằm nâng cao hiệu năng và trải nghiệm người dùng.

# ⚙️ Hướng dẫn cài đặt

### Clone project

```bash
git clone https://github.com/ilysm-Tqisme/SweetCakeShop.git
```

### Cấu hình Database

Chỉnh sửa chuỗi kết nối trong:

```text
appsettings.json
```

Sau đó chạy Migration:

```powershell
Update-Database
```

hoặc

```bash
dotnet ef database update
```

Cuối cùng nhấn **F5** để chạy dự án.
